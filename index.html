Software Freedom
and Open Source

I believe in Free software, not necessarily gratis, but free. If something is deemed 'Free software', the user can run, copy, distribute, study, change and improve the software. Free software is defined by the license.

Open Source software is different, if the license is restrictive, the user no longer has the freedom of doing what they want, just the source code.
